PLAYBOOK STAGE SPECIFICATION (INTEGRATION LAYER)
Research Context: Technology advisory firm (vendor-neutral, commission-based) synthesizing all research layers into actionable ABM playbooks. Each playbook targets a specific vertical x title combination with precision messaging, multi-touch sequences, and objection handling tailored to that intersection.
Output Requirements: Each playbook delivers a complete go-to-market toolkit for one vertical x title combination. Outputs are immediately actionable for sales and marketing teams executing ABM campaigns.

PLAYBOOK AGENT: INTEGRATION & SYNTHESIS
Core Question: How do we combine horizontal insights, vertical specifics, and title context into a cohesive, actionable playbook that enables precision outreach to this specific buyer segment?

Integration Vectors:

1.	Pain Points (Vertical x Title Specific)
o	How do title pain points manifest differently in this specific vertical?
o	What are the top 3-5 pain points for this title in this vertical specifically?
o	Which infrastructure advisory services address these pain points?
o	What urgency drivers exist at this intersection?
o	What language does this title in this vertical use to describe their challenges?

2.	Value Proposition (Adapted)
o	What is the primary value proposition for this vertical x title?
o	How does this differ from generic messaging?
o	What vertical-specific language should be used?
o	What title-specific outcomes should be emphasized?
o	What makes the advisory model uniquely valuable to this combination?

3.	Proof Points (Relevant Evidence)
o	What proof points resonate with this title in this vertical?
o	What case studies or metrics are most relevant?
o	What credentials or certifications matter?
o	What peer references would be compelling?
o	What quantified outcomes are credible for this audience?

4.	Outbound Sequence (Multi-Touch Campaign)
o	What is the optimal outreach sequence?
o	What channels does this title in this vertical prefer?
o	What content offers drive engagement at each stage?
o	What timing aligns with vertical buying cycles and title availability?
o	What triggers indicate right moment for outreach?

EXECUTION CONTEXT

Dependencies:
- Requires completed Layer 1 horizontal research (all 5 agents)
- Requires completed Layer 2 vertical research (all configured verticals)
- Requires completed Layer 3 title research (all configured title clusters)

Parallel Execution:
- All playbook agents execute in parallel after Layer 3 completion
- Each playbook agent independent (no cross-playbook dependencies)

Context Injection:
- Layer 1 context provides buyer journey and messaging foundation
- Layer 2 context provides vertical-specific regulations, pain points, buying patterns
- Layer 3 context provides title-specific decision authority, preferences, objections
- Playbook agent synthesizes all three into cohesive, actionable toolkit

Output Usage:
- Direct input to marketing campaign execution
- Sales enablement materials and training
- Content creation prioritization
- ABM platform targeting and sequencing

Methodology:
- Synthesize insights from all prior layers
- Create cohesive narrative specific to this combination
- Make concrete and immediately actionable
- Ground all recommendations in research
- Be specific, not generic - generic playbooks fail
- Include actual copy/scripts, not just recommendations
